# AI Customer Feedback Analyzer

A Python application that reads customer reviews from a CSV file, performs basic sentiment analysis, and generates output files with sentiment classifications and summary statistics.

## Features

* Reads reviews from CSV files
* Classifies reviews as Positive, Negative, or Neutral
* Generates sentiment analysis results in CSV format
* Creates a summary report with sentiment statistics

## Technologies

* Python
* CSV Processing
* File Handling

## Future Improvements

* Advanced sentiment analysis
* AI-powered review classification
* Data visualization
* API integration
